让系统在回答"介绍短线模式"时，**优先读取和使用《屠龙表 - 短线模式.csv》**的内容，而不是给出泛泛的通用回答。

   - 支持多种编码（utf-8、gbk、gb2312、utf-8-sig）

   - CSV文件获得+2分额外权重
   - "短线模式.csv"获得+3分额外权重

5. **强制工具调用** ✅
   - System prompt强制要求调用知识库工具

```python
short_term_keywords = ["短线模式", "短线", "模式", "打板", "低吸", "半路", "接力", "龙头"]
is_short_term_question = any(keyword in query for keyword in short_term_keywords)
```

### 2. 查询扩展
当识别到短线模式问题时，生成专门的查询变体：
```python
expanded = [

    "趋势接力主升模式",
    "纯趋势模式",
    "短线买卖点",
]
```

### 3. 权重提升算法
```python
if boost_csv and source.endswith('.csv'):

```
