# 智能股票分析Agent系统 - 完整版 🚀

## 🎯 系统概述

基于LangChain + 通义千问的智能交易分析系统，集成了：
- ✅ **增强RAG知识库** - 深度学习交易知识
- ✅ **图片识别分析** - 识别截图、理解图表 🆕
- ✅ **回撤风控专家** - 专注交易复盘和风控 🆕
- ✅ **实时数据获取** - 涨停股、行情、连板等
- ✅ **思维链推理** - 展示分析过程
- ✅ **短期记忆** - 记住最近5轮对话

## 🆕 最新功能

### 📸 图片识别功能（v3.0）

**新增能力：**
- 识别截图中的文字（OCR）
- 理解图片内容和数据
- 结合知识库深度分析图片
- 支持交易截图、复盘笔记、K线图等

**使用示例：**
```
💬 ~/Desktop/涨停板.png 今天这个涨停板怎么样？
```

**支持格式：**
- 本地图片：`/path/to/image.png`
- 网络URL：`https://...`
- 图片类型：PNG, JPEG, GIF, BMP, WebP

## 🎓 知识库说明

### ku/ 文件夹

系统从 `ku/` 目录加载交易知识：

```
ku/
├── 错题集龙.txt          # ⭐ 交易失败案例（优先加载）
├── 交易原则.txt          # ⭐ 核心交易原则（优先加载）
├── 战法.txt              # 操作战法
├── 情绪流.docx           # 情绪周期理论
├── 操作系统.txt          # 操作系统
├── 超短xxx.docx          # 超短交易系列
└── ...                   # 更多文档
```

**文档优先级：**
1. **必须加载**：错题集、风控、原则、卖点
2. **核心文档**：交易、操作、战法
3. **其他文档**：情绪、龙头、打板

**添加自定义知识：**
1. 将文档放入 `ku/` 目录
2. 支持 `.txt` 和 `.docx` 格式
3. 系统自动加载并索引

## 🐛 常见问题

**Q: 如何使用图片识别？**
A: 在提问时直接包含图片路径：
```
💬 /path/to/image.png 这个截图怎么看？
```

**Q: 图片识别失败？**
A:
1. 检查图片路径是否正确
2. 确认文件存在且可读
3. 验证图片格式（支持PNG, JPEG等）

**Q: 回答不够准确？**
A:
1. 使用风控版或完整版
2. 提问更具体
3. 添加更多知识文档到 `ku/`

**Q: 响应太慢？**
A:
1. 图片识别需要8-10秒
2. 如不需要图片，使用其他版本
3. 关闭重排序功能（修改代码）
