## 🔍 如果测试失败

### 症状1：回答是泛泛而谈，没有引用CSV
```
短线模式是一种交易策略，通常用于捕捉短期市场波动...
```

**解决方案：**
1. 确保使用了 `start_agent.sh` 或手动终止了旧进程
2. 在浏览器中清空聊天历史（点击"清空"按钮）
3. 重新提问

### 症状2：提到了CSV但内容不详细

**解决方案：**
1. 检查是否真的加载了CSV文件：
```bash
python test_csv_loading.py
```
2. 如果CSV文件没有加载，检查ku目录下是否有CSV文件

### 症状3：完全没有反应或报错

**解决方案：**
1. 检查API KEY是否正确
2. 查看终端日志，看是否有错误信息
3. 运行后端测试验证逻辑：
```bash
python test_agent_shortterm.py
```
